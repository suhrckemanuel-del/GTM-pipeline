# Value note — Lakera AI

**Vertical:** AI / LLM security  
**Geo:** EU — Switzerland  
**Momentum cited:** ~€18.2M Series A (Sifted); LLM security GTM; developer + security buyer paths.

## Observation

Security products that also sell to developers can **split messaging**: engineering wants SDKs and fast tests; security wants governance, reporting, and procurement-ready assurances. If both are blended on one page, each side sees half a product.

## Suggestion

Maintain **dual tracks** with a shared backbone story: developer path (integrations, CI hooks, quick eval) and security path (policy, dashboards, evidence for risk reviews). Add a one-page **“bring your CISO”** summary: what you log, what you block, and what artifacts export for audits.

## Expected impact

Faster dual-thread deals (Eng + Security) and fewer lost cycles from “cool demo, unclear enterprise fit.”
