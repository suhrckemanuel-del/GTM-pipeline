# Value note — Gleef

**Vertical:** AI / B2B SaaS  
**Geo:** EU — France (Paris)  
**Momentum cited:** Active AMSA GTM strategy project with founding team; live work on ICP, messaging, and channel priority across customer tiers.

## Observation

Early-stage B2B SaaS often wins the first deals with a sharp founder story, then slows when the same message is pushed into multiple tiers without re-segmenting proof, pricing, and sales assist. From the project work, the friction is less “more leads” and more **consistent tier-specific narrative** so outbound, website, and partner conversations do not contradict each other.

## Suggestion

Run a **two-week tier experiment**: pick one under-served tier, write a one-page “why now / why us” for that tier only, and route five outbound or inbound conversations through that doc. Track not just meetings but **objection patterns** (price, timing, competitor) to see if the bottleneck is message, ICP, or enablement.

## Expected impact

Faster learning on which tier to overload with GTM spend before scaling paid channels — fewer wasted demos and clearer handoff between founder-led and repeatable sales.
